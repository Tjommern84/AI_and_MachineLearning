Prosjektet består av fem hovedoppgaver og to plott-trinn:

Task 1: Filtrering av differensielt uttrykte gener (Exercise vs NSCLC)
Task 2: Pathway-analyse (GSEA/ORA) og “Opposite Pathways”
Task 3: Identifikasjon av immunrelaterte pathways
Task 4: Differensialanalyse av LUAD-data
Task 5: Maskinlæringspipeline (LogReg, MLP, K-Means)

Compare Biology: Sammenligning av biologiske og ML-funn
Plot-scripts: Genererer figurer for Task 1–4 og ML-sammenligning

⚙️ Kjøring

Fra prosjektroten:
python Group6_LungCancer_PyCharm.py

eller kjør filen direkte i PyCharm.

Alle resultater lagres automatisk i output/ og figurer i plots/.
